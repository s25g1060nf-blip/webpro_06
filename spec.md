### 図

追加・削除機能は期間限定が変わった時に活用する．

```mermaid
stateDiagram-v2
    [*] --> メニュー一覧表示
    メニュー一覧表示 --> 詳細表示（価格など）:上記の表を参考
    詳細表示（価格など） --> 編集画面: 編集ボタン
    編集画面 --> 登録
    登録 --> メニュー一覧表示
    メニュー一覧表示 --> 追加画面: 追加ボタン
    追加画面 --> 登録
    詳細表示（価格など） --> 削除画面: 削除ボタン
    削除画面 --> メニュー一覧表示
```

### 図

追加・削除機能は偏差値が変わった時や新たにできた時などに活用する．

```mermaid
stateDiagram-v2
    [*] --> 一覧表示
    一覧表示 --> 詳細表示: 学部，学部ごとの偏差値
    詳細表示 --> 編集画面: 編集ボタン
    編集画面 --> 登録
    登録 --> 一覧表示
    一覧表示 --> 追加画面: 追加ボタン
    追加画面 --> 登録
    詳細表示 --> 削除画面: 削除ボタン
    削除画面 --> 一覧表示
```
